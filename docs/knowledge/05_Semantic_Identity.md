# Mamascota — Semantic Identity

**Status:** Canonical
**Version:** 1.0
**Language:** English (master document)

# Primary entity

Mamascota

Entity type:

AI assistant for veterinary consultation preparation.

---

# Canonical definition

Mamascota is an AI assistant that helps people caring for dogs and cats prepare for veterinary consultations through structured conversations and clear communication with veterinary professionals.

---

# Core purpose

Prepare.

Organize.

Communicate.

Never diagnose.

---

# Primary domain

Veterinary consultation preparation.

---

# Secondary domains

- Animal health communication
- Veterinary reports
- Symptom organization
- Multilingual veterinary support
- Pet care assistance

---

# Entity attributes

Mamascota is:

- multilingual;
- AI-powered;
- structured;
- veterinary-oriented;
- educational;
- communication-focused;
- report-driven.

---

# Explicit exclusions

Mamascota is not:

- a veterinarian;
- a diagnostic system;
- a treatment recommendation system;
- a telemedicine platform;
- an emergency veterinary service.
